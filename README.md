1. Constant Prefix - TWS 
2. Package - package {{package}}
3. @since {{since}}
4. slug - tailwind-wp-starter


# steps
1. Clone and activate the plugin
2. You will see the Tailwind Admin menu
3. You have integrate your Tailwind UI components here
4. "src" is main folder
5. App.js is an entry point
6. Run `npm run start` for development
7. Run `npm run build` for production

PS> We are using WOrdPress's `wp-scripts` for development
